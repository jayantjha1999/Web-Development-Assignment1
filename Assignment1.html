4) Defines the document type and version of HTML being used. Represents the root element of an HTML document. Contains meta-information about the HTML document, such as title, links to stylesheets, and scripts. Sets the title of the HTML document displayed on the browser's title bar or tab. <meta> Provides metadata about the HTML document, such as character encoding, author, and viewport settings. <link> Defines the relationship between the current document and an external resource, commonly used for stylesheets. <script> Embeds or references an external script, such as JavaScript, within an HTML document. <body> Contains the content of the HTML document, including text, images, links, and other elements. <h1>, <h2>, <h3>, <h4>, <h5>, <h6> Represents headings of different levels, where <h1> is the highest and <h6> is the lowest level. <p> Defines a paragraph of text. <a> Creates hyperlinks, allowing users to navigate to another web page or resource. <img> Embeds an image in the HTML document. <ul> Defines an unordered list, typically displayed with bullet points. <ol> Defines an ordered list, displayed with numbers or letters as list markers. <li> Represents a list item within an ordered or unordered list. <table> Defines a table to organize data into rows and columns. <tr> Represents a table row. <th> Defines a table header cell, typically used in the first row or column of a table. <td> Defines a table data cell, containing the actual data within a table. <form> Creates an HTML form for user input, used to collect user data. <input> Defines an input field within a form, such as text, password, or submit buttons. <button> Represents a clickable button, commonly used within forms. Full-Stack Web Development Tech Stack

Frontend Technologies:
HTML (HyperText Markup Language): Used for structuring content on the web, defining elements and layout.
CSS (Cascading Style Sheets): Styles HTML elements, defining their appearance and layout on the web pages.
JavaScript: A versatile scripting language used to create dynamic and interactive web applications, enabling client-side interactivity.
Backend Technologies:
Node.js: A JavaScript runtime environment that allows developers to run JavaScript on the server-side, facilitating scalable and efficient backend development.
Express.js: A minimalist web application framework for Node.js, simplifying the creation of robust APIs and web applications.
Python: A versatile programming language often used for backend development, offering readability and ease of use.
Database Technologies:
MySQL: An open-source relational database management system, widely used for managing structured data.
MongoDB: A NoSQL database, storing data in flexible, JSON-like documents, allowing for seamless integration of unstructured data.
